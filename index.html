Eventster

A web app to help users create and manage events they are hosting.

Technology

Eventster uses an React front end, an Express/Node.js backend, and an MongoDB database. The app allows a user to create an event and allow other users to register for that event. Users registered to an event can comment on an event and get updates from the Event Organizer. Unauthenticated users can browse and search for events and view event details.

Responsibilities

As this was a group project, my primary responsibilties were authentication and the database. I used passport-js along with JSON Web Tokens for the authentication. I created schemas using mongoose for the Users, Events, and Comments to limit the results returned from various API calls to the database.

Casa

A web application that assists potential homebuyers in finding their dream home.

Technology

Casa is built with Handlebars and Bootstrap on the front end, an Express/Node.js backend, and an MySQL database. Casa allows a user to save home information using the Zillow API and then add custom notes for details that are important to them. Once those notes are stored in the database, a user can compare notes about the homes they have saved.

Responsibilities

As this was a group project, my primary responsibilties were authentication and the database. I used passport-js for the authentication. Sequelize was used as the objet-relational mapping tool to interface with the MySQL database.

Sports News Scraper

An web application to save sports articles from the New York Times.

Technology

The Sports News Sraper is built with ReactJS, Express/Node.js, and an MySQL. Users can scrape the New York Times Sports section to get new article titles and headlines. A user can then save those articles to be read at a later time. Notes can be added to save articles.
